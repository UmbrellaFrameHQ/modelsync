# Examples

Bu klasor ModelSync paketlerini hizlica denemek icin kopyalanabilir ornekler icerir.

| Ornek | Aciklama |
|---|---|
| [MySQL quickstart](mysql-quickstart.md) | MySQL/MariaDB icin modelden tablo SQL'i uretme ve calistirma |
| [SQL Server quickstart](sqlserver-quickstart.md) | SQL Server icin tablo olusturma akisi |
| [SQLite in-memory](sqlite-in-memory.md) | Testlerde SQLite `:memory:` kullanimi |
| [DestructiveOperationOptions](destructive-operation-options.md) | Drop/alter gibi riskli islemler icin explicit opt-in |

Her ornekte once NuGet paketi yuklenir, sonra model tanimlanir ve generator kullanilir.
