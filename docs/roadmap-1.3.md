# ModelSync 1.3 Roadmap

The following features are intentionally out of scope for 1.2.1 and are candidates for 1.3.

## Index Modeling

- Composite indexes.
- Included columns.
- Filtered indexes.
- Sort direction control.
- Clustered/non-clustered index selection.
- Expression indexes.

## Provider API

- Richer provider-specific index option objects.
- Stronger typed default/check expression helpers.
- Expanded analyzer rules for provider-specific schema APIs.
